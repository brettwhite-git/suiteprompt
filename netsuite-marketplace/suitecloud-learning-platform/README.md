# SuitePrompt

A repository for NetSuite prompts, templates, and skills, combined with a learning platform for SuiteCloud development.

## Overview

SuitePrompt serves two purposes:

1. **Marketplace**: A curated collection of NetSuite prompts, templates, and skills organized by functional area
2. **Learning Platform**: An interactive learning platform for SuiteCloud development

## Marketplace

- **Prompts**: Ready-to-use NetSuite prompts for various business functions
- **Skills**: Comprehensive skill sets for NetSuite development and administration
- **Templates**: Reusable templates for common NetSuite tasks
- **Search & Filter**: Find prompts and skills by category, function, or keyword

### Categories

**Prompts**: Accounting, Sales, Inventory, Manufacturing, Field Service, Support, Forecasting, MCP

**Skills**: NetSuite Administration, SuiteScript Development, SuiteCloud Platform, Customization & Configuration, Integration & API Usage, Reporting & Analytics

## Learning Platform

- **Modular Learning Paths**: Functional Users, Developers, or Advanced tracks
- **Rapid 101**: Get started in 15 minutes
- **Feynman Technique**: Simple explanations that build to technical details
- **Visual Learning**: Interactive diagrams, animations, and workflows
- **Progress Tracking**: Track your learning progress

### Learning Paths

- **Functional Users → Developers**: For NetSuite users learning development concepts
- **Developers → NetSuite**: For developers learning NetSuite-specific concepts
- **Advanced Topics**: Deep dives for experienced developers

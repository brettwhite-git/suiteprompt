## Key Concepts

### 1. Customization vs Configuration

- **Configuration**: Using NetSuite's built-in features (like setting up workflows, creating custom fields through the UI)
- **Customization**: Writing code to create new features or modify existing behavior

### 2. Why SuiteCloud?

- Automate repetitive tasks
- Create custom reports and dashboards
- Integrate with other systems
- Build custom user interfaces
- Extend NetSuite's functionality

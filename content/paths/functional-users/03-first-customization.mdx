## Step-by-Step: Creating a Custom Field

### Step 1: Navigate to Custom Fields

1. Go to **Customization > Lists, Records, & Fields > Entity Fields**
2. Click **New**

### Step 2: Configure Your Field

Let's create a "Customer Priority" field:

- **Label**: Customer Priority
- **Type**: List/Record
- **List/Record**: Create a new list with values:
  - High Priority
  - Medium Priority
  - Standard
- **Help Text**: Indicates the service level for this customer
- **Default Value**: Standard

### Step 3: Set Display Settings

- **Display Type**: Normal
- **Applies To**: Customer
- **Subtab**: Classification (or create a custom subtab)

### Step 4: Set Access Permissions

- **Store Value**: Checked
- **Available Externally**: Checked if needed for APIs
- **Role Restrictions**: Set which roles can see/edit this field

### Step 5: Save and Test

1. Click **Save**
2. Navigate to any Customer record
3. Look for your new field!

## Advanced: Custom Record Types

Once you're comfortable with custom fields, you can create entirely new **Custom Record Types**. These are like creating brand new tables in your database.

Example use cases:
- Project Management: Custom "Project" records
- Real Estate: Custom "Property" records
- Education: Custom "Course" records
